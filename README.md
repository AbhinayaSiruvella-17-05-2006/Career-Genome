Career Genome

Career Genome is an AI-powered career development platform that helps users analyze their career readiness, identify skill gaps, practice interviews, and generate learning roadmaps.
The system combines data analysis, AI assistance, and career intelligence tools to guide users toward better career outcomes.

🚀 Features

User Authentication

User Signup and Login

JWT based authentication

Secure password hashing with bcrypt

Career Readiness Scanner

Upload resume (PDF)

Compare resume with job description

Calculate readiness score

Peer percentile benchmarking

Skill Gap Analyzer

Detect missing skills for a target role

Generate personalized learning roadmap

Provide learning topics and mini projects

Interview Practice Avatar

AI-based interview question simulator

Role-based questions

Instant scoring and feedback

Failure Intelligence Engine

Analyze career setbacks

Identify failure patterns

Suggest actionable improvement strategies

Career Shock Alerts

Track layoffs and hiring trends

Identify emerging technology skills

Monitor job market signals

AI Career Assistant

Chat-based mentor for career questions

Technical explanations and learning guidance

Project Generator

Generate portfolio project ideas

Based on role, skills, and learning goals

Admin Dashboard

View database collections

Monitor stored career analysis results

Manage platform data
